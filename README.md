## 🚀 Quick Start

1. Clone the repo
```bash
git clone https://github.com/yourusername/school-navigator.git
```

2. Install dependencies
```bash
npm install
```

3. Configure environment variables
```bash
cp .env.example .env
```

4. Launch! 🚀
```bash
npm run dev
```
